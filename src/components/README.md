#  Hospital Management Dashboard

A responsive Hospital Management Dashboard built using **React** and **Material UI**.

##  Features
- Appointments listing with patient details
- Diagnosis management page
- Patient info & vitals cards
- Previous visits tracking
- Clean UI with Material UI components
- Fully component-based architecture

##  Tech Stack
- React
- Material UI (MUI)
- React Router
- JavaScript (ES6)

##  How to Run
```bash
npm install
npm start
